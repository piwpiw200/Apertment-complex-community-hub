# Apartment Community Frontend

A frontend-only React + Tailwind web app for an apartment complex community.

## Features

- Resident and admin login flow
- Resident join request and admin approval
- Community feed with photo/video-style posts
- Event creation and participation
- Meeting launch and live meeting indicator
- Group and direct chat simulation
- Issue reporting for apartment issues
- Admin controls for approvals and community operations

## Setup

```bash
npm install
npm run dev
```

Open the app at `http://localhost:5173`.

## Build

```bash
npm run build
```

## Notes

This project is frontend-only and does not include a backend or database. State is simulated in-memory.
